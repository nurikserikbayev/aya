# AYA — website

Landing page for **AYA** (آية, "miracle") — the AI navigation brain that turns any
cheap robot into an autonomous supply carrier: map-free, GPS-optional, commanded in
plain language. For farms, and non-lethal frontline logistics.

Single self-contained `index.html` (inline CSS/JS, no build step, no dependencies),
deployed via **GitHub Pages**.
